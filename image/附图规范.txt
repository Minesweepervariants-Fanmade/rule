规则附图文件名规范:
    1.采用规则的所有代称按照列表顺序加入下划线作为文件名, 后缀.png
    2.由于系统限制不允许文件名使用? * < > / \ :
        所以做出以下转义
        -q ?
        -a *
        -l <
        -g >
        -s /
        -b \
        -c :
        -- -
    示例:
        2G*包含名称
        id = "2G*"
        name = "Group*"
        name.zh_CN = "互异四连块"
        那么他的文件名为"2G-a_互异四连块_Group-a.png"
